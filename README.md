turing machine emulator
====


## Descriotion

L = {1, 0 , x}
w = {1, 0}
S = {0,1,2..}

it accepts as follow wxw, e.g. (0100w0010, 1w1, 000w000)


## Requirement
cargo  
rust lang 


## Install
```
cargo build
cargo test
```
