turing machine emulator
====


## Descriotion

L = {1, 0 }
w = {1, 0}
S = {init, reject, a,b....}

it accepts as follow ww, e.g. (01000010, 11, 000000)


## Requirement
cargo  
rust lang 


## Install
```
$cargo build
$cargo test or cargo run
```
